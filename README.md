# HexStack Claude Marketplace

Official marketplace for Claude Code plugins and extensions by HexStack.

## Available Plugins

### Claude Grimoire
- **Version**: 1.0.0
- **Type**: Native Application
- **Description**: Manage Claude Code configurations with a native GUI

## Installation

1. Install Claude Code
2. Add this marketplace: `claude marketplace add https://hexstack.app/claude/marketplace.json`
3. Browse available plugins: `claude marketplace list`
4. Install a plugin: `claude plugin install claude-grimoire`

## Publishing

To publish this marketplace:
1. Host the contents of this directory on a web server
2. Ensure marketplace.json is accessible via HTTPS
3. Share the marketplace URL with users
